# Variable Mapping Table

## English

| Fish Tank Variable | Economic Variable | Interpretation |
| --- | --- | --- |
| Fish Tank | Global economic system | The bounded system where all variables interact |
| Fish | People, companies, banks, nations, investors | Actors competing for survival and resources |
| Fish food | Money supply, credit, subsidies, QE, stimulus, resource allocation | Policy and financial inputs injected into the system |
| Water quality | Trust, institutional stability, transparency, social confidence | The invisible condition that determines whether actors believe the system works |
| Oxygen | Liquidity, energy, supply chain capacity | The circulating capacity required for survival and transaction flow |
| Water temperature | Inflation, interest rates, geopolitical pressure | Macro pressure that changes actor behavior and survival thresholds |
| Overfeeding | Asset bubble, debt expansion, leverage, greed, moral hazard | Too much input creates waste, distortion, and dependency |
| Underfeeding | Unemployment, recession, demand collapse, social unrest | Too little input weakens survival, confidence, and production |
| Dirty water | Corruption, opacity, misinformation, broken trust | The system becomes hard to read and unsafe to participate in |
| Fish panic/death | Bank runs, bankruptcies, social instability, financial crisis | Visible collapse behavior after hidden fragility accumulates |

## 繁體中文

| 魚缸變數 | 經濟變數 | 解釋 |
| --- | --- | --- |
| 魚缸 | 全球經濟系統 | 所有變數互相作用的邊界系統 |
| 魚 | 人民、企業、銀行、國家、投資者 | 爭取生存與資源的行動者 |
| 魚飼料 | 貨幣供給、信用、補貼、量化寬鬆、刺激政策、資源投放 | 注入系統的政策與金融輸入 |
| 水質 | 信任、制度穩定、透明度、社會信心 | 決定參與者是否相信系統仍可運作的隱性條件 |
| 氧氣 | 流動性、能源、供應鏈承載能力 | 維持生存與交易流動所需的循環能力 |
| 水溫 | 通膨、利率、地緣政治壓力 | 改變行為與生存門檻的宏觀壓力 |
| 過度餵食 | 資產泡沫、債務擴張、槓桿、貪婪、道德風險 | 過多輸入會製造浪費、扭曲與依賴 |
| 餵食不足 | 失業、衰退、需求崩塌、社會不安 | 輸入不足會削弱生存、信心與生產 |
