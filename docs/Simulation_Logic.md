# Simulation Logic

## English

The simulation logic tracks interaction among ecological variables rather than single-variable prediction.

Basic loop:

1. Define the tank state.
2. Assign food input, oxygen level, water quality, temperature, and fish behavior.
3. Observe whether resource allocation improves survival or creates waste.
4. Track trust degradation when water quality declines.
5. Track panic behavior when oxygen, food, or water quality cross stress thresholds.
6. Score systemic fragility using the risk assessment model.
7. Compare crisis scenarios against governance interventions.

The model should be interpreted as a symbolic systems layer. It is useful for visualizing instability patterns, not for producing exact forecasts.

## 繁體中文

此模擬邏輯追蹤生態變數之間的互動，而不是進行單一變數預測。

基本循環：

1. 定義魚缸狀態。
2. 設定飼料投放、氧氣水平、水質、水溫與魚群行為。
3. 觀察資源分配是否改善生存，或製造浪費。
4. 當水質下降時追蹤信任退化。
5. 當氧氣、飼料或水質跨越壓力門檻時追蹤恐慌行為。
6. 使用風險評估模型計算系統脆弱性。
7. 將危機情境與治理干預進行比較。

此模型應被理解為象徵性的系統層。它適合用於視覺化失衡模式，而不是產生精準預測。
